This program runs a File sharing system. It does not takes any arguments. File that needs to be shared among 5 peers needs to be mentioned in the server main method.
This Program will only run for 5 Clients. You can Change the value from 5 to any other value by changing the Client Class. 
Note: File input and Number of Clients is static and not Dynamic
Way to RUN 
store it in any drive 
Compile:
javac cn/*.javac
Go inside cn folder and Type these commands
To RUN server 
java Server
To run Client
Open multiple commandPrompt to run 5 clients simultaneously 
java Client
